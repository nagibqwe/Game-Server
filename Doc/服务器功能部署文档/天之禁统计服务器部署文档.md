- jdk1.8

### 应用对应目录
- 统计服使用 StatLogServer

###  应用结构
-StatLogServer 根目录
	-resources  存放配置目录
		-config.properties 配置文件
	-sql 存放sql目录
		-tzj_stat_log.sql sql文件
	-startStatLogServer.bat window 执行批处理脚本
	-startStatLogServer.sh linux执行脚本
	-StatLogServer-1.0-SNAPSHOT.jar 应用程序

### 配置统计库 和 gm数据库 地址
-配置路径 resources/config.properties

```
#gm平台数据库地址九零  一起玩 www .90 175.com
gm.jdbc.url=jdbc:mysql://127.0.0.1:3306/gm?useUnicode=true&characterEncoding=utf8&zeroDateTimeBehavior=convertToNull&useSSL=true&serverTimezone=GMT%2B8
#m平台用户名
gm.jdbc.username=root
#gm平台密码
gm.jdbc.password=root


#统计库mysql 地址
stat.jdbc.url=jdbc:mysql://127.0.0.1:3306/tzj_stat_log?useUnicode=true&characterEncoding=utf8&zeroDateTimeBehavior=convertToNull&useSSL=true&serverTimezone=GMT%2B8
#统计库 用户名
stat.jdbc.username=root
#统计库 密码
stat.jdbc.password=root
```
#linux启动 关闭
-./startStatLogServer.sh start 启动命令
-./startStatLogServer.sh stop 关闭命令
